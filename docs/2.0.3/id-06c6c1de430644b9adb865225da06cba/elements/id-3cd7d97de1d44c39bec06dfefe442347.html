JCDR ()

Standaard
Juriconnect Decentrale Regelgeving (JCDR) is een afgesproken tekstvolgorde (syntaxis) voor verwijzingen naar decentrale regelgeving in de voorziening Decentrale Regelgeving en Officiële Publicaties (DROP).
StatusVerplicht
Functioneel toepassingsgebiedJCDR moet worden toegepast op elektronische verwijzingen naar decentrale regelgeving.

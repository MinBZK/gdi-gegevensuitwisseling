Begrippen > afleidingsregel
DefinitieEen regel die beschrijft hoe bepaalde soorten gegevens kunnen worden afgeleid uit andere soorten gegevens.
ToelichtingEen transformatieregel is ook een afleidingsregel, waarbij de regel de gegevens in een gegevensmodel van een doel afleidt uit gegevens in een gegevensmodel van een bron.

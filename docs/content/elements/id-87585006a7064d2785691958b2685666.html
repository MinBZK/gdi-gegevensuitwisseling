DefinitieDe Geography Markup Language is een XML-gebaseerde vorm voor het uitdrukken van schema's voor geografische gegevens.
StatusAanbevolen
BeheerorganisatieOGC
Linkhttps://www.ogc.org/standard/gml/
Functioneel toepassingsgebiedGeo-standaarden moeten worden toegepast op de uitwisseling van geografische informatie tussen organisaties, waarbij de ruimtelijke dimensie van significant belang is.

DefinitieDe veranderingen van een object.
Redactionele notitieDeze definitie is gebaseerd op NEN 3610:2022, maar die bevat geen formele definitie.
ToelichtingDit vraagt het vastleggen van de tijdstippen van veranderingen van eigenschappen, de toestand, van een object in de werkelijkheid. De periode waarin een object eenzelfde toestand had is de geldigheidsperiode.

DefinitieRDF is een standaard voor het uitwisselen van gegevens op het web. Als partijen data als ‘RDF-Triples’ uitwisselen, wordt het voor de ontvangende partij makkelijker om de data geautomatiseerd te interpreteren en te linken aan andere data(sets).
BeheerorganisatieW3C
URLhttps://www.forumstandaardisatie.nl/open-standaarden/Rdf

Functioneel toepassingsgebiedGestructureerd via het Web ter beschikking stellen van gegevens door overheden, zodat de aanbieder of overige partijen (geautomatiseerd) deze gegevens kunnen koppelen.

DefinitieEen samengestelde verzameling metadata over datasets en dataservices.
Redactionele notitieDeze definitie is gebaseerd op de definitie van Catalog in DCAT 2.0, maar specifiek gemaakt voor datacatalogi.
ToelichtingEen Catalog in DCAT 2.0 is algemener van aard en kan allerlei bronnen bevatten. In DCAT 2.0 is een Catalog ook een Dataset.

Standaarden > NEN 3610
DefinitieHet Basismodel geo-informatie (NEN3610) bevat de termen, definities, relaties en algemene regels voor de uitwisseling van informatie over ruimtelijke objecten.
TypeStandaard

BeheerorganisatieGeonovum

Functioneel toepassingsgebiedGeo-standaarden moeten worden toegepast op de uitwisseling van geografische informatie tussen organisaties, waarbij de ruimtelijke dimensie van significant belang is.

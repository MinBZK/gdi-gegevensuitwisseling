MTOM ()
Message Transmission Optimization Mechanism of kortweg MTOM is een methode om op efficiënte wijze binaire data naar en van webservices te versturen.
StatusAanbevolen
Functioneel toepassingsgebiedHet verzenden van grote hoeveelheden data (bijvoorbeeld attachments) in SOAP-berichten.
BeheerorganisatieW3C

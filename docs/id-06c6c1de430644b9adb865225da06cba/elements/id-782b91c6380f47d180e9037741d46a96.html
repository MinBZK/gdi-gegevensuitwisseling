Standaardiseren NL GOV profile for CloudEvents en AsyncAPI ()
De standaarden "NL GOV profile for CloudEvents" en AsyncAPI zijn een belangrijke basis voor gebeurtenisgedreven gegevensuitwisseling, maar hebben op dit moment geen formele status binnen de overheid. Het sturen op het gebruik hiervan is wenselijk om te komen tot meer standaardisatie. De standaarden zouden aangeboden moeten worden aan Forum Standaardisatie zodat deze overheidsbreed aanbevolen of verplicht worden.

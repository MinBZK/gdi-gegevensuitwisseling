Een ding, een tastbaar iets, in de werkelijkheid, zoals daarnaar gekeken wordt vanuit een bepaald domein.
IDobject
Redactionele notitieMerk op dat MIM 1.2 de term object in de toelichting verwart met een instantie van een objecttype. Dat is echter geen object in de werkelijkheid maar een object in een administratie. Om die reden is dat deel van de toelichting in MIM weggelaten.
ToelichtingMet in de werkelijkheid wordt bedoeld dat het om de daadwerkelijke onderwerpen van gesprek gaat, de verzameling van de concrete tastbare dingen waarover we het hebben. Bijvoorbeeld, de persoon Jan, Paleis 't Loo.

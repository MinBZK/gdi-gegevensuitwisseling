formele historie ()
De veranderingen van gegevens in een registratie
IDformele historie
Redactionele notitieDeze definitie is gebaseerd op NEN 3610:2022, maar die bevat geen formele definitie.
ToelichtingDeze historie kan pas in de registratie zelf worden bepaald. Het betreft typisch een startmoment van de registratie en een eindmoment van de registratie. Verder kan ook een versienummer worden gebruikt om de wijzigingen van een record in tijd te kunnen vastleggen. De periode waarin gegevens in de registratie aanwezig en actueel waren is de registratieperiode.
